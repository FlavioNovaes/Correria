# 🏍️ Correria

Projeto desenvolvido em **HTML, CSS e JavaScript**, criado para auxiliar motoboys e entregadores no controle de **ganhos**, **gastos** e **corridas realizadas** no dia a dia.

O sistema permite registrar informações das entregas, acompanhar valores recebidos e visualizar dados importantes da rotina de trabalho de forma simples e prática.

Este projeto está sendo desenvolvido como parte dos estudos em JavaScript, focando principalmente em manipulação do DOM, lógica de programação e armazenamento de dados no navegador.

## 🚀 Funcionalidades

- Cadastro de corridas realizadas
- Registro de ganhos por entrega
- Controle de gastos:
  - ⛽ Combustível  
  - 🛠️ Manutenção  
  - 🍔 Alimentação  
  - 📦 Outros custos  
- Cálculo automático de lucro
- Histórico de registros
- Interface simples, moderna e responsiva
- Armazenamento de dados no navegador com LocalStorage
